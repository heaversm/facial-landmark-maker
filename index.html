Face Landmark Maker

Generates Facial Landmarks based off the IBUG 300-W dataset model for images in which a face cannot be detected. This is useful in machine learning and computer vision algorithms which require facial landmarks to be present, such as DLib and openCV. I built this after generating some non-human faces with pix2pix that I wanted to use with FaceMorph

INSTRUCTIONS: Place the images you want to make landmarks for in the 'assets' folder. To load a specific image, put the filename with extension in the URL 'image' parameter, e.g. 'http://localhost:8000/?image=myimage.png'. Drag the points to fit the image. Then hit 's' to save out the repositioned points to a text file corresponding to your image name.
